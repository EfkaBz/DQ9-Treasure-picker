# DQ9-Treasure-Picker
DQ9 Treasure Picker est un petit outil PC conçu pour Dragon Quest IX – Les Sentinelles du Firmament, permettant de visualiser et sélectionner facilement les cartes au trésor du jeu à partir d’une bibliothèque d’images locales.  Ce logiciel a été développé pour simplifier la recherche et l’identification des cartes.
